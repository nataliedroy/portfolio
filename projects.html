Things I've Done

This project was an assignment for INFO201 Spring 2020 quarter. The assignment was a group project to create a data visualization website revolving around a specific dataset.

Working with three other people, me and my team decided to focus on environmental data, specifically revolving around global warming and emissions. Using multiple datasets, we worked together to create three different interactive graphs.

This project was programmed in R and turned into a web app through Shiny. While we worked together on many aspects of the site, some of my main responsibilities were creating and designing the look of the website.
